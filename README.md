# 🪔 RiMit Layers — Business Management Platform

> AI-powered business operations platform managing a real e-commerce business
> across **Etsy**, **eBay**, and a custom storefront.

## What This Project Does
- 📦 Tracks orders and inventory across all 3 platforms
- 🔔 Instant Telegram + email alerts for orders and buyer messages
- 🤖 AI-powered reply drafts and sales insights (Claude API)
- 💰 Full fee and expense tracking for tax filing
- 📊 Side-by-side dashboard comparing all platforms
- 🛒 Live storefront with Stripe payments

## Tech Stack
**Backend:** Python 3.12, FastAPI, SQLAlchemy, SQLite, APScheduler
**Frontend:** React 18, Tailwind CSS
**Website:** Next.js 14
**DevOps:** Docker, GitHub Actions, Terraform, Grafana, Prometheus
**AI:** Claude API (Haiku), Ollama (local dev)
**Notifications:** Telegram Bot, Gmail SMTP

## Status
🚧 Currently in active development — built in public.
